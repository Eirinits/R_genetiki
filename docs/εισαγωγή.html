Εφαρμογές της R στη γενετική

2 Εισαγωγή

Aρχικά, μπορούμε να χρησιμοποιήσουμε την R και την κονσόλα του ΡΣ για να εκτελέσουμε βασικές μαθηματικές πράξεις:

3 + 5
## [1] 8
12 / 7
## [1] 1.714286

Παρόλα αυτά, για να μπορέσουμε να κάνουμε πιο ενδιαφέροντες και χρήσιμες αναλύσεις πρέπει να αναθέσουμε διάφορες τιμές σε ένα αντικείμενο ή μεταβλητή.

Για να δημιουργήσετε ένα αντικείμενο χρειάζεστε:

  • ένα όνομα (π.χ. “weight_kg”)
  • μια τιμή (π.χ. «1»)
  • το σύμβολο άναθεσης “<-” (assignment symbol) Στο σενάριό σας, χρησιμοποιώντας το σύμβολο ανάθεσης R “<-”, αντιστοιχίστε το “23” στο αντικείμενο “weight_kg”:
weight_kg <- 23

Στη συνέχεια, εκτελέστε αυτήν τη γραμμή κώδικα στο σενάριό σας. Μπορείτε να εκτελέσετε μια γραμμή κώδικα πατώντας το κουμπί “Run” που βρίσκεται ακριβώς πάνω από την πρώτη γραμμή του σεναρίου σας ή μπορείτε να χρησιμοποιήσετε την κατάλληλη συντόμευση:

Συντόμευση εκτέλεσης Windows/Linux: Ctrl + Enter Συντόμευση εκτέλεσης Mac: Cmd (⌘) + Enter

Για να εκτελέσετε πολλές γραμμές κώδικα, μπορείτε να επιλέξετε όλες τις γραμμές που θέλετε να εκτελέσετε και στη συνέχεια να πατήσετε "Run ή να χρησιμοποιήσετε τις παραπάνω συντομεύσεις.

Αφού δημιουργήσουμε το αντικείμενο και βρίσκεται πλέον στη μνήμη της Ρ, μπορούμε να το καλέσουμε και να το χρησιμοποιήσουμε σε διάφορες αριθμητικές πράξεις.

weight_kg * 2.2
## [1] 50.6

Μπορούμε επίσης να αλλάξουμε τη τιμή ενός αντικειμένου αναθέτοντάς του μία διαφορετική τιμή

weight_kg <- 50

Μπορούμε να χρησιμοποιήσουμε, επίσης, υπάρχοντα αντικείμενα για να δημιουργήσουμε καινούργια.

weight_lb <- 2.2 * weight_kg

2.1 Ονομασία

Ακολουθούν ορισμένες σημαντικές λεπτομέρειες σχετικά με την ονομασία αντικειμένων στην R.

  • Αποφύγετε κενά και ειδικούς χαρακτήρες: Τα ονόματα αντικειμένων δεν μπορούν να περιέχουν κενά ή το σύμβολο μείον (-). Μπορείτε να χρησιμοποιήσετε το "_" για να κάνετε τα ονόματα πιο ευανάγνωστα. Θα πρέπει να αποφύγετε τη χρήση ειδικών χαρακτήρων στο όνομα του αντικειμένου σας (π.χ.! @ #. Κ.λπ.). Επίσης, τα ονόματα αντικειμένων δεν μπορούν να ξεκινούν με έναν αριθμό.
  • Χρησιμοποιήστε σύντομα, εύκολα κατανοητά ονόματα: Θα πρέπει να αποφύγετε την ονομασία των αντικειμένων σας χρησιμοποιώντας μεμονωμένα γράμματα (π.χ. «n», «p», κ.λπ.). Αυτό είναι κυρίως για να σας ενθαρρύνει να χρησιμοποιήσετε ονόματα που θα ήταν λογικά σε οποιονδήποτε διαβάζει τον κωδικά σας (ένας συνάδελφος, ή ακόμα και εσείς έναν χρόνο από τώρα). Επίσης, η αποφυγή υπερβολικά μεγάλων ονομάτων θα κάνει τον κωδικό σας πιο ευανάγνωστο.
  • Αποφύγετε τα κοινά χρησιμοποιούμενα ονόματα: Υπάρχουν πολλά ονόματα που ενδέχεται να έχουν ήδη ορισμό στη γλώσσα R (π.χ. «mean», «min», «max»). Μια ένδειξη ότι ένα όνομα που επιλέξατε υπάρχει ήδη ως ένα function, είναι ότι εάν αρχίσετε να πληκτρολογείτε το συγκεκριμένο όνομα στο RStudio και παίρνει μια έγχρωμη επισήμανση ή το RStudio σας δίνει μια προτεινόμενη αυτόματη συμπλήρωση, έχετε επιλέξει ένα όνομα που χρησιμοποιείται από τις βασικές λειτουργίες της R ή κάποιου πακέτου.

2.2 Εντολές / Συναρτήσεις (Functions)

Με τον όρο functions αναφερόμαστε σε «σενάρια» που αυτοματοποιούν πιο περίπλοκα εντολές και λειτουργίες. Πολλές λειτουργίες συμπεριλαμβλάνονται στην Ρ ενώ πιο εξειδικευμένες μπορούν να διατεθούν με την εισαγωγή πακέτων R. Μια συνάρτηση συνήθως παίρνει ένα ή περισσότερα ορίσματα(inputs) και συχνά (αλλά όχι πάντα) επιστρέφουν μια τιμή (output). Ένα τυπικό παράδειγμα θα ήταν η εντολή sqrt(). Το όρισμα της εντολής πρέπει να είναι ένας αριθμός και η τιμή που επιστρέφεται είναι η τετραγωνική ρίζα αυτού του αριθμού.

sqrt(5)
## [1] 2.236068
sqrt(weight_kg)
## [1] 7.071068

Άλλες εντολές δέχονται παραπάνω από ένα όρισματα, όπως για παράδειγμα η εντολή round().

round(8.3692)
## [1] 8
round(8.3692, digits = 1)
## [1] 8.4
